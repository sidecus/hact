# HACT Output Checklist

Run this checklist before finalizing output.

- Scope is project-specific (`cwd` filtered) unless user asked for global.
- Evidence window is stated (session count + date range).
- Inferences are separated from explicit user statements.
- Principles are concise, non-overlapping, and actionable.
- Execution protocol enforces principles without redundancy.
- Proposed doc edits do not conflict with existing core guardrails.
- User-facing content redacts personal names and local path identifiers when requested.
- Residual uncertainty is stated when evidence is partial.
